01

What is Matrix?

Matrix is an open protocol for real-time communication — think of it like email, but for chat. Anyone can run their own server, and servers talk to each other automatically. You keep your account and conversation history no matter which server you're on.

It supports end-to-end encrypted direct messages, group rooms, voice and video calls, and works with many different client apps.

02

How to sign up

  1. Open app.element.io in your browser, or install the Element app.
  2. Click Create account.
  3. Keep the default server (matrix.org) or choose another — it doesn't matter, you can reach anyone on Matrix regardless of server.
  4. Fill in a username and password, then verify your email if prompted.
  5. You're in. Use the address below to find me and say hi.
